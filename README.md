# Machine-Learning-Journey
Welcome to my Machine Learning learning repository.
A collection of Machine Learning concepts, implementations, experiments, and projects built while learning ML from fundamentals to advanced topics.
This repository contains my hands-on implementation of Machine Learning concepts, algorithms, and projects. The goal is to build a strong understanding of both the mathematical foundations and practical applications of Machine Learning.

## 📚 Topics Covered

- Linear Regression
- Gradient Descent
- Polynomial Regression
- Regularization (Ridge, Lasso, ElasticNet)
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Trees
- Random Forest
- Support Vector Machine (SVM)
- Clustering (K-Means, Hierarchical)
- Principal Component Analysis (PCA)
- Model Evaluation Techniques
- Cross Validation
- Hyperparameter Tuning
