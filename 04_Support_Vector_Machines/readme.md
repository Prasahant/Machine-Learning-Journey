This repository is part of my **Machine Learning Journey**, where I learn machine learning concepts from scratch by combining theory, mathematics, handwritten notes, and practical implementations.

In this section, I explored **Support Vector Machines (SVM)**, one of the most powerful supervised machine learning algorithms used for both **classification** and **regression**. The focus of this repository is to build a strong conceptual understanding of how SVM works and how different kernels help solve non-linear classification problems. SVM aims to find the optimal hyperplane that maximizes the margin between different classes. :contentReference[oaicite:0]{index=0}

## 💡 Key Takeaways

- SVM performs well on both linear and non-linear datasets.
- Support vectors are the most influential data points.
- The Kernel Trick allows SVM to work in higher-dimensional spaces without explicitly creating new features.
- Polynomial Kernel implicitly models polynomial relationships.
- RBF Kernel is generally the default choice for complex non-linear datasets.
- Proper feature scaling is important before training an SVM model.
